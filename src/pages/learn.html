What is Computer Science?

Computer Science is the study of computation and information—covering algorithms, data, hardware, software, and systems.

Fields

AI & Data

Machine learning, data analysis, natural language processing, and computer vision.

Web & Apps

Frontend and backend development, APIs, and mobile applications.

Starter Resources

  • MDN Web Docs: Pro-level docs for web fundamentals—HTML, CSS, JavaScript, APIs.
