Tools & Resources

Set up a professional toolkit early and use it in every project.

  • Editor: VS Code
  • Version control: Git & GitHub
  • Terminal: Bash or PowerShell basics
  • Package managers: pip / npm
  • Learning hubs: MDN Web Docs, Python docs, FreeCodeCamp
  • Practice: LeetCode/HackerRank for problem-solving
